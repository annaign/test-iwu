## Задача 4. Country Suggest: компонент подсказки названия страны с флагами

### Installation

Install with npm:

```
npm install
npm run start
```

Install with yarn:

```
yarn install
yarn start
```

### Build

Build with npm:

```
npm run build
```

Build with yarn:

```
yarn build
```
